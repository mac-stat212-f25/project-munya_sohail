# Salaries/Careers in Data Science


>What is this GitHub repository all about?
 
This repository analyzes salary levels across data-science job postings.
It includes:
A dataset of job postings (CSV, shp file),
The qmd's clean the data,
currency conversions,
computation salary distributions;
computation of averages by industry, seniority, job title, location;
visualizations of salary distributions,
A qmd report that produces a final summary
